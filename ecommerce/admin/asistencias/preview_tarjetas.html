📋 Cómo Usar el Sistema

  1. Generar Tarjetas: Acceder al botón "🎫 Tarjetas PDF" en el módulo de asistencias
  2. Imprimir: Descargar el PDF e imprimir en papel de buena calidad
  3. Plastificar: Opcional pero recomendado para mayor durabilidad
  4. Escanear: Usar la interfaz "📱 Escanear Asistencia" con un lector de códigos
  5. Registro Automático: El sistema registra la asistencia al instante
